# CourseFlow - מערכת ניהול קורסים

## סיכום הפרויקט

זה פרויקט של מערכת לניהול ושיווק קורסים דיגיטליים בעברית, בשם CourseFlow. המערכת מאפשרת למורים ויוצרי קורסים ליצור קורסים חדשים, לעצב פליירים שיווקיים, ליצור דפי נחיתה, וכל זה עם ממשק משתמש מודרני וידידותי.

---

## הקבצים ושמותיהם החדשים

הקבצים הורשמו עם שמות משמעותיים שמשקפים את תכנם:

### 1. **dashboard.html** (code.html קודם)
- **תיאור**: דשבורד הקורסים הראשי
- **תפקיד**: מציג את כל הקורסים של המשתמש, מאפשר חיפוש וסינון, ולחצן ליצירת קורס חדש
- **תכונות**:
  - רשימת קורסים קיימים בתצוגת גריד
  - חיפוש וסינון לפי קטגוריה
  - מעבר לדף יצירת קורס חדש

### 2. **landing-page.html** (code1.html קודם)
- **תיאור**: דף נחיתה שיווקי לקורס
- **תפקיד**: הצגת תצוגה מקדימה חיה של דף הקורס בשלב השלישי
- **תכונות**:
  - בנייה אוטומטית של דף נחיתה על סמך נתוני הקורס
  - תצוגה מקדימה בזמן אמת
  - תיקייה ממשק ללא קוד (עריכה חזותית)

### 3. **flyer-designer.html** (code2.html קודם)
- **תיאור**: עורך פלייר שיווקי
- **תפקיד**: עיצוב וערוך פליירים שיווקיים לקורסים בשלב השני
- **תכונות**:
  - עריכה מהירה של פרטי הקורס
  - עדכון אוטומטי של התצוגה
  - הורדה וחלוקה של הפלייר

### 4. **course-creation.html** (code3.html קודם)
- **תיאור**: טופס יצירת קורס
- **תפקיד**: איסוף נתונים בסיסיים ותצורת עיצוב בשלב הראשון
- **תכונות**:
  - טופס מפורט לנתוני קורס
  - בחירת עיצוב וסגנון
  - שמירת הנתונים למעבר בין דפים

## תקשורת בין הדפים

### מנגנון ה-localStorage

כל הדפים משתמשים ב-**localStorage** כדי לתקשר ולשתף נתונים:

```javascript
// שמירת נתוני קורס
const courseData = {
    title: 'שם הקורס',
    description: 'תיאור',
    price: '₪450',
    // ... תכונות נוספות
};
localStorage.setItem('courseData', JSON.stringify(courseData));

// קריאת נתוני קורס
const savedData = JSON.parse(localStorage.getItem('courseData'));
```

### זרימת הנתונים

```
Course Creation (course-creation.html)
            ↓ [שמור נתונים]
         Flyer Designer (flyer-designer.html)
            ↓ [קרא נתונים + שמור]
         Landing Page (landing-page.html)
            ↓ [קרא נתונים + הצג]
         Dashboard (dashboard.html)
```

## ניווט בין הדפים

### מסלול הזרימה:

1. **Dashboard** (דשבורד ראשי)
   - לחצן "צור קורס חדש" → **Course Creation**
   - לחצני "פתח קורס" → **Course Creation**

2. **Course Creation** (יצירת קורס - שלב 1)
   - לחצן "הבא לשלב פלאייר" → **Flyer Designer** (משמר נתונים)
   - לחצן "חזור" → **Dashboard**

3. **Flyer Designer** (עיצוב פלייר - שלב 2)
   - לחצן "הבא: דף נחיתה" → **Landing Page** (קורא ומשמר נתונים)
   - לחצן "חזור" → **Course Creation**

4. **Landing Page** (דף נחיתה - שלב 3)
   - לחצן "שמור וסיים" → **Dashboard** (קורא נתונים)
   - לחצן "חזור" → **Flyer Designer**

## תכונות טכניות

### 1. **Responsive Design**
- מעובד לעבודה בשולחן עבודה וטלפון
- משתמש ב-Tailwind CSS למעצב
- תמיכה בעברית (RTL - Right-to-Left)

### 2. **Dark Mode**
- תמיכה מלאה בארבעת הדפים
- שינוי צבעים אוטומטי בהתאם להעדפות המערכת

### 3. **צבעים וטיפוגרפיה**
- צבע ראשי: `#13ecda` (תירקום)
- טיפוגרפיה: Inter + Heebo + Noto Sans Hebrew
- Material Design Icons לאייקונים

### 4. **שמירת מצב**
- כל הנתונים נשמרים ב-localStorage
- אפשרות לכתוב טיוטה ולהמשיך אחר כך

## איך להשתמש

### פתיחת הפרויקט:
1. פתח את `dashboard.html` בדפדפן
2. לחץ על "צור קורס חדש"
3. מלא את פרטי הקורס בעמוד היצירה
4. לחץ "הבא" כדי להמשיך לעיצוב הפלייר
5. אפשר לערוך את הפלייר ולהמשיך לדף הנחיתה
6. צפה בתצוגה מקדימה של דף הקורס
7. לחץ "שמור וסיים" לחזרה לדשבורד

### איחסון נתונים:
- הנתונים נשמרים בזיכרון המתכנן של הדפדפן (localStorage)
- אם תנקה את היסטוריית הדפדפן, הנתונים יימחקו
